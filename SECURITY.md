# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 14.x.x  | :white_check_mark: |
| 11.x.x  | :x:                |
| 10.x.x  | :x:                |
| 9.x.x   | :x:                |
| 8.x.x   | :x:                |
| < 8.0   | :x:                |

## Reporting a Vulnerability

Please report (suspected) security vulnerabilities to [d4rk.s0ul](https://discord.com/users/682983233851228161/) via Discord. You will receive a response from us within 24 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity but historically within a few days.
